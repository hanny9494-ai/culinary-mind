# Resources  —  External Data Sources


## Updates (2026-04-04)
- FoodAtlas (gjorgjinac/foodatlas) selected as CRITICAL bridge layer: 230K food-compound relations, MIT license, maps ~1,000 foods to ~28,000 chemical compounds from FooDB. Difficulty: LOW (Python package, clean API)
- USDA FoodData Central rated CRITICAL for L2a: 380K+ foods, free API, public domain, provides quantitative parameters (protein%, fat%, moisture%, amino acid profiles, minerals) that L0 boundary_conditions reference
- Wikidata SPARQL endpoint (CC0) rated CRITICAL for L6 translation layer: QIDs serve as universal entity linker bridging FoodOn, USDA, and FooDB ontologies
- FlavorGraph (Sony AI): 6,653 ingredients, 1,525 flavor compounds — rated HIGH value for the FT (flavor) layer
- FoodOn Ontology (30K classes, OWL, CC BY 3.0) rated HIGH for L2a ingredient taxonomy

## Updates (2026-04-04)
- FoodAtlas（230K 食物-化合物关系，MIT 许可）是关键桥接层：提供 Ingredient→Compound 链接，配合 L0 的 Compound→ScientificPrinciple 链接，难度低，Python 包有干净 API
- Wikidata SPARQL 端点（CC0）可用于 L6 翻译层和通用实体链接器（QID 桥接 FoodOn、USDA、FooDB）
- FlavorGraph（Sony AI，6,653 种食材，1,525 种风味化合物）对 FT（风味）层价值高
