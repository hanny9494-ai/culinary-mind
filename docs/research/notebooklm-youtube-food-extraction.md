# Research: NotebookLM + YouTube for Food Knowledge Extraction

**Date**: 2026-03-27
**Researcher**: researcher agent
**Purpose**: 评估 NotebookLM 和替代方案用于从中文烹饪视频中提取食材/技法知识

---

## NotebookLM 限制

| 资源 | 限制 |
|------|------|
| Notebooks/账号 | 100 个 |
| Sources/Notebook | **300 个** |
| YouTube 视频 | 通过字幕处理，不看画面 |
| 中文字幕 | 支持（简/繁），自动生成字幕可能不准 |
| 无字幕视频 | **不能处理** |
| 视频时长上限 | ~2-3 小时（无硬限） |
| 日查询限制 | ~50-100 次/天（免费层，未公布硬限） |
| **API** | **不存在**（截至 2026-03） |
| Plus 版 | $19.99/月（Google One AI Premium），更多查询，仍无 API |

**结论：NotebookLM 只能手动用，不能自动化。适合 Jeff 个人探索，不适合 pipeline。**

---

## 推荐方案：yt-dlp + Gemini API

| 步骤 | 工具 | 成本 |
|------|------|------|
| 1. 批量抽字幕 | yt-dlp | 免费 |
| 2. 筛选有价值视频 | Gemini Flash（文本） | <¥7 |
| 3. 高价值视频深度提取 | Gemini Pro File API（上传视频） | ¥200-1200 |

```bash
# yt-dlp 批量抽中文字幕
yt-dlp --write-sub --write-auto-sub --sub-lang zh-Hans,zh-Hant,zh \
       --skip-download --sub-format srt \
       --output "%(playlist_title)s/%(title)s.%(ext)s" \
       PLAYLIST_URL
```

Gemini File API：支持上传视频（视觉+音频），最长 1 小时，但不接受 YouTube URL — 必须先下载。
